# ICGRIS Dashboard (Frontend Demo)

Stack: Next.js (App Router) + Tailwind CSS + TypeScript

## Quick Start (Local)
```bash
npm i
npm run dev
```

## Deploy on Vercel
1. Push this folder to a Git repo (GitHub).
2. Import the repo in Vercel.
3. Use default build settings (Next.js).
